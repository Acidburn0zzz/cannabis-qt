    Benvingut a la pàgina de Cannabis-qt

    Cannabis-qt és un senzill programa de gestió d'un local social on es pugui dispensar cànnabis mitjançant un control estricte.

    El programa està fet amb les llibreríes QT (http://qt.nokia.com)

    La web del programa : http://karasu.github.com/cannabis-qt

    El programa permet:

    - Gestionar els socis
- Gestionar els pots de cànnabis
- Gestionar el consum de cànnabis dels socis
- Gestionar les aportacions dels socis
- Controlar els ingressos de l'associació
- Imprimir informes d'ingressos
- Importació i exportació de les dades (SQLITE)
